# Measure Formatting

| Measure | Format |
|---|---|
| Total Rent Due | `$#,0;($#,0);-` |
| Total Rent Collected | `$#,0;($#,0);-` |
| Outstanding Rent | `$#,0;($#,0);-` |
| Collection Rate | `0.0%` |
| Payment Count | `#,0` |
| Rent Collected YTD | `$#,0;($#,0);-` |
| Rent Collected Prior Year | `$#,0;($#,0);-` |
| Rent Collected YoY % | `0.0%` |
| Lease Count | `#,0` |
| Active Lease Count | `#,0` |
| Monthly Contracted Rent | `$#,0;($#,0);-` |
| Annualized Contracted Rent | `$#,0;($#,0);-` |
| Leases Ending | `#,0` |
| Maintenance Request Count | `#,0` |
| Open Maintenance Requests | `#,0` |
| Estimated Maintenance Cost | `$#,0;($#,0);-` |
| Actual Maintenance Cost | `$#,0;($#,0);-` |
| Maintenance Cost Variance | `$#,0;($#,0);-` |
| Average Resolution Days | `0.0` |
| Completed Maintenance Requests | `#,0` |
| Budget Revenue | `$#,0;($#,0);-` |
| Budget Expense | `$#,0;($#,0);-` |
| Budget NOI | `$#,0;($#,0);-` |
| Rent to Budget Variance | `$#,0;($#,0);-` |
| Rent to Budget Variance % | `0.0%` |
| Property Count | `#,0` |
| Tenant Count | `#,0` |

Set fact numeric columns to **Do not summarize** so report authors use governed measures rather than implicit aggregations.
