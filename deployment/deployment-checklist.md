# Deployment Checklist

## Pre-Deployment

- Pull request approved
- Python validation passed
- SQL scripts reviewed
- Required documentation updated
- No secrets committed
- Sample data available
- Validation rules passed

## Deployment

- Promote Dev to Test
- Run validation in Test
- Confirm semantic model relationships
- Confirm report visuals
- Promote Test to Prod

## Post-Deployment

- Validate latest pipeline run
- Confirm row counts
- Confirm Power BI refresh or Direct Lake connectivity
- Confirm dashboard pages
- Record deployment notes
